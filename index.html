Features

What makes our pixel bot smart

1. Smart Attack Algorithm


Unlike other pixel bots, our pixel bot offers a smarter monster-search algorithm. This allows the bot to prioritize and queue monsters when attacking.

2. Smart Loot Function


The smart loot function enables the bot to pick up all loots dropped by the monster except those dropped by the monster killed by another player.

3. Auto-Pots


The pixel bot also heals himself whenever his hp drops to a certain percentage. In this way, the pixel bot lasts longer on the field.

4. Random Walk


The pixel bot also features a smart random-walk algorithm. The feature allows the bot to search for walkable spots visible on the map.

5. Auto-Teleport


The bot also features an auto-teleport algorithm. If the bot cannot find a monster for a given period of time, the bot will automatically teleport to find better spots.

Setup Guide

An easy step-by-step guide on how to setup your pixelbot

1. Create a copy of Ragnarok Online Folder

Create a duplicate (Copy) of your Ragnarok Online Folder and rename it into Ragnarok Online - Pixel

2. Go to the Ragnarok Online - Pixel folder

Navigate into the Ragnarok Online - Pixel folder and open data.grf using GRF Editor.

3. Open data.grf using GRF Editor

4. Merge preset grf(s) into data.grf

On GRF Editor app, Select Edit -> Merge
